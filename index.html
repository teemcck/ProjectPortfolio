Side Projects

WolverineSoft 2024 Pumpkin Game Jam

48 hour game jam submission following the theme "Block Party." This game is a 2-player dynamic rhythm-based game where players synchronize actions to musical beats to outperform their opponent. There are 3 unique tracks, a regular and halloween them, and 9 unique characters to select from. This project was designed using C#11 inside of the Unity Game Engine.

WolverineSoft 2024 Turkey Game Jam

48 hour game jam submission following the theme "Counterproductive." This game is an asymetical, cooperative, infinite runner. Each game consists of a turkey, and a builder, who work together to progress as far as possible. This project was created using Blender 4.3 for models and animations, and was programmed using C#11 inside the Unity Game Engine.

WolverineSoft 2024 Mini Game Jam

This 24 hour game jam was my first formal introduction to Unity and C#. Unlike typical game jams, all teams worked together to create one project, "WolveyWare," a parody of "WarioWare." Each person created a WarioWare like minigame to all be compiled together. My minigame, "Hide your Sins," was one of the games included. It was designed using C#11 inside of the Unity Game Engine.
